# Problem
Imagine you are playing a game of Mafia with family friends. One person must take on the role as the host/moderator, 
but a few issues can occur where:

1) No one wants to moderate
2) The moderator forgets roles or vote counts
3) The moderator is bias
4) Not enough people to play

# Solution
Mafia solves these issues by having the mobile app act as the host/moderator. The app has core features like:
- Custom Game Settings
- Random Role Assigning
- Announcements
- Vote Counting
- Game Management

Being an application, there is a lack of bias and "forgetting" as information is stored. 
However, bear in mind that there is a lack of moderation for cheating, so make sure the group of friends you play with are trustworthy.
