# Mafia Values
All the design decisions will be based on these core values.
1) **Simplicity**
2) **Accessibility**
3) **FOSS (Free Open Source Software)**

## Simplicity
Make the most with the least. The app's sole purpose is to solve the issues mentioned in [problem.md](docs/problem.md).
Therefore, excessive features that do not complement the solution should not be added and bloated code should be avoided.

## Accessibility
Ideally, everyone can access this solution. Realistically, some trade-offs made may limit access (e.g an app requires a mobile device). 
Regardless, the app should have a wide reach to as many people by making it work offline and available on Google Play. 
If successful, the app will be made available on iOS App Store.

## FOSS (Free Open Source Software)
This repository remains open for anyone to fork, contribute, or just look at the code under the [MIT License](LICENSE.MD). 
The project remains free and is funded through donations on [FUNDING.yml](.github/FUNDING.yml).
